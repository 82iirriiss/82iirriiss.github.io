# 파이썬 코딩 시, 알아야 할 꿀팁!

## 날짜 입력하기

    - datetime.datetime.now() 보다는, timezone.now() 사용을 추천함
    - current_year = timezone.now().year

## __str__() 메소드

    - 객체를 string 으로 표현해 주기 위해 모든 클래스에 정의 할 수 있다.
    - Question 객체를 알아보기 쉬운 스트링으로 표현하기 위해 self.question_text를 __str__() 함수에서 리턴한다.
